## BizFloとは？
**BizFlo** は、中小企業・チーム向けの **オールインワン業務管理**。  
顧客・案件・KPI・会計・受発注を一元管理できます。

### 主な機能
- 👥 顧客管理：紹介者紐づけ、名刺OCR、LINE連携
- 📂 案件管理：ステータス・履歴・コピー
- 📊 KPI：目標/実績の可視化
- 💰 会計：仕訳連動、PL/BS自動作成
- 🔄 受発注：受注/発注の一元化、帳票生成
- 🛠 運用：タスク/カレンダー/操作ログ
